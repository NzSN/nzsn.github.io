4. The system \(\lambda 2\)

\(\mathbb{T}2 = \mathbb{V} | (\mathbb{T}2 \rightarrow \mathbb{T}2) | (\Pi\mathbb{V}:*. \mathbb{T}2)\), with \(\mathbb{V}\) a set of type variables.

\(\Lambda_{\mathbb{T}2} = V | (\Lambda_{\mathbb{T}2}\Lambda_{\mathbb{T}2}) | (\Lambda_{\mathbb{T}2}\mathbb{T}2) | (\lambda V : \mathbb{T}2 : \Lambda_{\mathbb{T}2}) | (\lambda\mathbb{V}:*. \Lambda_{\mathbb{T}2})\)

There are

  • Object variables
  • first order application
  • first order abstraction

Extra second order terms

  • second order application
  • second order abstraction (\(\mathbb{V}\) is type variables)

from left to right respectivly.

  • Statement. \(M : \sigma\) where \(M \in \Lambda_{\mathbb{T}2}\) and \(\sigma \in \mathbb{T}2\)
  • Declaration. A statement with a term variables or a type variable as subject.
  • \(\emptyset\) is a \(\lambda\text{2-context}\).
  • If \(\Gamma\) is a \(\lambda\text{2-context}\), \(\alpha \in \mathbb{V} \wedge \alpha \notin dom(\Gamma)\), then \(\Gamma, \alpha: *\) is a \(\lambda\text{2-context}\) .
  • If \(\Gamma\) is a \(\lambda\text{2-context}\), \(\rho \in \mathbb{T}2\) such that \(\alpha \in dom(\Gamma)\) for all free type variables in \(\rho\) and if \(x \notin dom(\Gamma)\), then \(\Gamma,x : \rho\) is a \(\lambda\text{2-context}\) .

\(\Gamma \vdash x : \sigma\) if \(\Gamma\) is a \(\lambda2\) context and \(x : \sigma \in \Gamma\) .

(form) \(\Gamma \vdash B:*\) if \(\Gamma\) is a \(\lambda2\) context, \(B \in \mathbb{T}2\) and all free type variables in B are declared in \(\Gamma\)

A term M in ΛT2 is called legal if there exists a λ2-context Γ and a type ρ in T2 such that Γ \(\vdash\) M : ρ .

4.1. List of derivation rule

  • (var) \(\Gamma \vdash x : \sigma\) if \(\Gamma\) is a \(\lambda2\) context and \(x : \sigma \in \Gamma\)
  • (appl) \(\frac{\Gamma \vdash M : \alpha \rightarrow \beta; \Gamma \vdash N: \alpha}{\Gamma \vdash MN : \beta}\)
  • (abst) \(\frac{\Gamma, x : \sigma \vdash M : \tau}{\Gamma \vdash \lambda x: \sigma . M : \sigma \rightarrow \tau}\)
  • (form) \(\Gamma \vdash B:*\) if \(\Gamma\) is a \(\lambda2\) context, \(B \in \mathbb{T}2\) and all free type variables in B are declared in \(\Gamma\)
  • (\(appl_{2}\)) \(\frac{\Gamma \vdash M : \Pi \alpha : * A; \Gamma \vdash B : *}{\Gamma \vdash MB : A[\alpha := B]}\)
  • (\(abst_{2}\)) \(\frac{\Gamma, \alpha : * \vdash M : A}{\Gamma \vdash \lambda \alpha : *. M : \Pi \alpha : *. A}\)
