2. Order Relations

A relation \(\leq\) is an partially order on X if it’s:

  • Reflexive
  • Transitive

  • Antisymmetric: \((x \leq y) \land (y \leq x) \Rightarrow x = y\)

    Let \(\leq\) is a partially order on set X, then the pair \((X, \leq)\) is called a partially ordered set.

    In addition, if

    \(\forall x, \forall y: (x \leq y) \lor (y \leq x)\)

    then \(\leq\) is called totally order on X and \((X, \leq)\) is a totally ordered set.

2.0.1. Order On Sets

With order relation on a set, we are able to build order on a set. There are some concepts:

X is a set and \(A \subseteq X\)

  • Upper bound: An element \(s \in X\) with \(s \leq a, \forall a \in A\)
  • Lower bound: An element \(s \in X\) with \(s \geq a, \forall a \in A\)

  • Least upper bound:

    Least upper bound of a set A is write as

    \(sup(A) := min\{s \in X; s is upper bound of A \}\)

  • Greatest lower bound:

    Greatest lower bound of a set A is write as \(inf(A) := max\{i \in X; i is lower bound of A \}\)

  • Maximum: An element \(s \in A\) that is upper bound of A.
  • Minimum: An element \(s \in A\) that is lower bound of A.

2.0.2. With Functions

  1. Monotone

    With order on domain and codomain we able to define order of functions.

    Let \(X := (X, \leq)\) and \(Y := (Y, \leq)\) be partially ordered sets, and \(f: X \rightarrow Y\) a function.

    if \(x_1 \leq x_2 \Rightarrow f(x_1) \leq f(x_2)\), we said \(f\) is increasing, strictly if \(f(x_1) < f(x_2)\)

    if \(x_1 \leq x_2 \Rightarrow f(x_1) \geq f(x_2)\), we said \(f\) is decreasing, strictly if \(f(x_1) > f(x_2)\).

  2. Bounded

    A function \(f: X \rightarrow Y\) is bounded, bounded upper or bounded below if im(f) is bounded, bounded upper or bounded below.

    if \(X\) is also a partially ordered set, then f is called bounded on bounded sets if, for each bounded subset \(A \subseteq X\), the restriction \(f|A\) is bounded.
