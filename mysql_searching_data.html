2. SELECT Command

Syntax: 

  SELECT what
  FROM table or tables
  WHERE conditions data must satisfy;

If you are not retrieve data from table you can ignore <FROM> and <WHERE>, such as 

  SELECT 2+2, 'Hello, world', VERSION();

when you want to retrieve data from table you need it: 

  SELECT * FROM student;
  SELECT name, id FROM student;
  SELECT * FROM score WHERE score > 95;

Complete support operator in <WHERE> is show in Page 643.

3. Post processing your search result

3.1. Order

   SELECT what
   FROM tables
   WHERE cond
   ORDER BY column1 [ASC/DESC], column2 [ASC/DESC]

   Note: ASC & DESC is optional. ASC default.

Order on multiple column is allow. 假设有一张数据表存放着一个年级所有学生的成绩我们希望可以根据班级，成绩的顺序排序那么可以: 

   SELECT * FROM student_score
   ORDER BY class, score;

这样获得的数据表中，相同班级的学生成绩会组成一个块，在这个块中又按成绩排序.

3.1.1. NULL

升序排列中NULL排在第一位， 降序排列中NULL排在最后一位.

3.2. Result number limit

只显示查询到的结果的前三个. 

   SELECT * FROM table ORDER BY column LIMIT 3;

3.3. Problems relate to Date

Support Date Operate:

  • Order
  • Search a day or a range of days.
  • Get year, month, day from Date.
  • Compute distance between two date.
  • Get another Date by plus distance with a Date.

For short, only show <WHERE> in following examples.

Examples: 

   WHERE date = '2008-10-01';
   WHERE date >= '1970-01-01 AND death < '1980-01-01';
   WHERE YEAR(birth) = 2020 AND MONTH(birth) = 12 AND DAYOFMONTH(birth) = 3;

计算时间间隔的方法有TIMESTAMPDIFF(), TODAYS() 

   SELECT last_name, first_name, birth, death,
   TIMESTAMPDIFF(YEAR, birth, death) AS age
   FROM president WHERE death IS NOT NULL
   ORDER BY age DESC LIMIT 5;

通过加减一个日期获得另一个日期的方法有DATEADD(), DATESUB. 

   SELECT DATE_ADD('1970-1-1', INTERVAL 10 YEAR);
   SELECT DATE_ADD('1970-1-1', INTERVAL 10 MONTH);
   SELECT DATE_ADD('1970-1-1', INTERVAL 10 DAY);
   SELECT DATE_SUB('1970-1-1', INTERVAL 10 YEAR);
