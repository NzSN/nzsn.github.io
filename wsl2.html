2. How to update WSL2 kernel ?

2.1. Methods

  • 从 kernel.org 下载最新的内核，配合微软的配置，从头编译安装最新的；
  • 从 WSL2 git 下载微软改好的内核，但是仍需要编译安装；
  • 下载微软弄好但是没发给Windows Update的内核。

2.2. 第三种方式的步骤

  • 找到微软发布的内核更新安装文件 msi https://www.catalog.update.microsoft.com/Search.aspx?q=wsl
  • 下载同 GIT上最新版本一致的 wslupdatex64.msi ；
  • 彻底关闭当前的 WSL，更新内核 —— 以管理员身份运行 wsl –shutdown；运行第2步下载的文件，完成内核更新；
  • 再次开启 WSL，检测内核版本 —— 以管理员身份运行 wsl， uname -a 。

3. How to migrate WSL2 to another disk ?

wsl --shutdown
wsl --export Ubuntu E:\Ubuntu.tar
wsl --unregister Ubuntu
wsl --import Ubuntu E:\Ubuntu.tar E:\WSL\
ubuntu.exe config --default-user xxx

4. Return idle memory to Host

in .wslconfig 

[wsl2]
pageReporting = <bool>    # Enable or disable the free memory page reporting feature (default true).
idleThreshold = <integer> # Set the idle threshold for memory compaction, 0 disables the feature (default 1).

or do it manually by run following command as the root user: 

echo 1 > /proc/sys/vm/compact_memory
