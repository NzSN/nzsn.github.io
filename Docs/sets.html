Sets is a concept to descript collection of objects.

With Logic, a Set can be devided into subsets. For example,

\(\mathbb{X} := \{ x : \text{x is cat} \}\) then \(\mathbb{A} := \{ x \in \mathbb{X}; \text{x is red cat} \}\)

\(\mathbb{X}\) is set of all cats. With propositional logic, I get a subset of cats which is a set of read cat.

With sets we are able to describe a class of things, entities, objects. Based on sets, able to describe relations, operations between classes precisely.

1. Relations between Sets

  • \(\subseteq\) or \(\subset\)
  • \(\supseteq\) or \(\supset\)

Both \(\subseteq\) and \(\supseteq\) is reflexive, transivetive.

1.1. Remark

  • Empty set implies every property.

    \(\emptyset_{\mathbb{X}} := \{ x \in \mathbb{X} | x \neq x \}\) then have property:

    \(x \in \emptyset_{\mathbb{X}} \Rightarrow E(x)\), where \(x \in \mathbb{X}\)

    With this property have \(x \in \emptyset_{\mathbb{X}} \Rightarrow x \in \emptyset_{\mathbb{Y}}\) which implies that \(\emptyset_{\mathbb{X}} \subseteq \emptyset_{\mathbb{Y}}\) and vice versa \(\emptyset_{\mathbb{Y}} \subseteq \emptyset_{\mathbb{X}}\). Hence \(\emptyset_{\mathbb{X}} = \emptyset_{\mathbb{Y}}\), which implies that empty set is unique.

  • Suppose \(\mathbb{X}\) is a set, then so is its power set \(\mathbf{P(\mathbb{X})}\), written as \(2^{\mathbb{X}}\). \(\mathbf{P(\mathbb{X})}\) is the set of all subset of \(\mathbb{X}\).

2. Operations

  • \(\cap\), intersection
  • \(\cup\), union

Above two operation are commutative, associative, distributive.

  • \(\backslash\), complement

  • \(\times\), cartesian product

    \(\displaystyle\prod^{n}_{j=1} \mathbb{X_{j}} := \mathbb{X}_1 \times \mathbb{X}_2 \times ... \times \mathbb{X}_N\)
