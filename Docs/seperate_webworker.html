7. <2025-05-19 Mon>

What I want to do is to let WebWorkers work on different Isolate. Maybe WebWorkers alread work on different Isolate for each instance of WebWorkers currently.

What I oberserve is total memory used by all WebWorkers does not allow to exceed 4GB which is a upper bound of allow memory of a single JS Heap.

Conceptually, different WebWorkers has different Isolate hence different V8 Context and JS Heap. Since Javascript is a single-thread language so if multiple instance of Script shared the same JS Heap then the those Script will trap into Race Condition. Which should not the case so I think different WebWorkers should use different JS Heap but all accounted as Total Used Memory and this value should not passed single JS Heap’s memory limit.
