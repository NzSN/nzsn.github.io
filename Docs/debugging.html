1. gdb/lldb debugging

1.1. How it work ?

Debugger mainly use following two things to debugging target.

  • debug info
  • ptrace

With debug info debugger can determine where is a symbol reside in source code and stop on which instruction.

For example there is a program: 

int work() {
  return 0;
}

int main() {
  work();
  return 0;
}

and debuginfo section of this program is: 

main.o:     file format elf64-x86-64
