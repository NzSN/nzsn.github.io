1. JS don’t reclaims all unused memory

1.1. What

Unused memory is not been release back to OS.

1.2. When

After update Chromium version from 114 (V8 version 11.4.183.8) to 124 (V8 version is 12.4.254.12)

1.3. Why

Still don’t know. Maybe it’s cause of GC does not return back those memory that been collected to OS.
