1. Construction of Integers

The motivation to construct Integers is to extend Addition of Naturals to be a Group Operator. Let’s look at why Addition of Naturals not a Group Operator

By definition of Groups, it’s operation should has property that \[ \forall g \in \mathbb{G}, \exists g^b \in \mathbb{G}: g \odot g^b = e\]

In Naturals we have \(e = 0\). Then let \(N \ni g = 1\) and if Naturals is a group t- hen we should be able to find out \(g^b\) such that \(g \odot g^b = 0\). In fact, we are unabl- e to do that cause there is not element because \[1 + \mathbb{N} = \mathbb{N} \backslash \{0\}\].

Obviously, Addition of Naturals is associatetive and has identity element. So The reason is it doesn’t not satisfied the third property of Group Operator. T- hen need to extend Naturals so that each of elements in Naturals has negative.

Then the most straight solution is to extend \[ \mathbb{N} \] so it able to represent \[ g^b \]. Intu- tively, have \[ \]\[ \forall n \in \mathbb{N}: n + (0 - n) = 0 \] and this can be written as \[ \forall n \in \mathbb{N} : (n - 0) + (0 - n) = n - n = 0 \] so we can build a set, \[ \mathbb{N}^2 \], that each element, \[ (x,y) \], represent \[ x - y \]. The name of the set is Integer.

1.1. Definition of Integers

Let \[\mathbb{Z} := (\mathbb{Z},+,\cdot)\] be Integers and \[(\mathbb{Z}, +)\] is a Group. \[\mathbb{Z}\] is defined as

\[ \mathbb{Z} := \mathbb{N}^{2} \backslash \sim \] with Equivalence Relation, \[ \sim \], that defined as \[ (m,n) \sim (m',n') := m + n' = m' + n\] Addition is defined as \[ [(m,n)] + [(m',n')] := (m - n) + (m' - n') = (m + m') - (n + n') = [(m+n',n+n')] \] Multiplication is defined as \[ [(m,n)] \cdot [(m',n')] := (m - n) \cdot (m' - n') = (mm' + nn') - (mn' + m'n) = [(mm'+nn', nm'+m'n)] \]

The Function \[ f: \mathbb{N} \rightarrow \mathbb{Z}, n \mapsto [(n,0)] \] is injective which implies that \[ \mathbb{N} \] is inclusion in \[ \mathbb{Z} \] and operation of \[ \mathbb{Z} \] is compatible with \[ \mathbb{N} \].

Let \[ \mathbb{Z} \] be a commutative ring with unity, 1, and without zero divisor. Operation on \[ \mathbb{Z} \] are restrict to usual operation on \[ \mathbb{N} \]. Let \[ g: \mathbb{Z} \rightarrow \mathbb{N} \], injective, be a function such that \[ g \mid \mathbb{N} = \text{(inclusion of N in Z)} \]. This implies that uniqueness of \[ \mathbb{Z} \] up to isomorphism.

2. Construction of Rational

But \[ \mathbb{Z} \] still not a Fields so next to extend \[ \mathbb{Z} \] so that we have a field struct- ure that extended from \[ \mathbb{N} \].

The reason why \[ \mathbb{Z} \] can not be a Fields similar to why Naturals is not a group, due to abcense of negation of elements with respect to multiplication in \[ \mathbb{Z} \].

So solution is similar, need to identifying \[ \frac{x}{y} \] from \[ \mathbb{Z} \] and by 8.7 Remarks, we have

\[ \mathbb{Q} := (\mathbb{Q} := \mathbb{Z}^{2}/\sim, +, \cdot) \] where \[ (x,y) \sim (x',y') := \frac{x}{y'} = \frac{x'}{y} \]

Addition: \[ [(x,y)] + [(x',y')] := [(xy' + x'y, y \cdot y')] \]

Multiplication: \[ [(x,y)] \cdot [(x',y')] := [(xx',yy')] \]

Obviously, \[ \mathbb{Q} \] is a field.

The function \[ f: \mathbb{Z} \rightarrow \mathbb{Q}, z \mapsto [(z, 1)] \] is injective and operation on \[ \mathbb{Q} \] is comp- atible with \[ \mathbb{Z} \]. Which implies that \[ \mathbb{Z} \subseteq \mathbb{Q} \], each \[ z \in \mathbb{Z} \] is identifying with its image under this function.

Due to both \[ \mathbb{Q} \] and \[ \mathbb{Z} \] are both countable, so there exists a function \[ g: \mathbb{Q} \rightarrow \mathbb{Z} \] such that \[ g \mid \mathbb{Z} = \text{inclusion of \mathbb{Z} in \mathbb{Q}} \]. which implies that unique- ness of \[ \mathbb{Q} \] up to isomorphism.

  • 1 Prop. \[ r \in \mathbb{Q} \iff \exists (p,q) \in \mathbb{Z} \times \mathbb{N^{\times}} : \frac{p}{q} = r \] Proof. Let \[ \mathbb{Q} \ni r = \frac{p}{q} \] . If \[ r < 0 \land q < 0 \] then we can rewrite r as \[ \frac{(-1)p \in \mathbb{Z}}{(-1)q \in \mathbb{N^{\times}}} \]. This implies the claim of this proposition is true \[ \blacksquare \]

  • 2 Prop. \[ \mathbb{Z} \] and \[ \mathbb{Q} \] are countably infinite.

    Proof. Let \[ f: \mathbb{N} \rightarrow \mathbb{Z} \] be an injective function such that

    \[ f(n \in \mathbb{N}) = \begin{cases} n/2 & \quad \text{if n is even} \\ -(n+1)/2 & \quad \text{if n is odd} \end{cases}\]

    Hence, \[ \mathbb{Z} \] is countable. So \[ \mathbb{Z} \times \mathbb{N^{\times}} \] is countable too which implies that \[ \mathbb{Q} \] is countable too due to there exist injective function (by (1 Prop) in this section) between \[ \mathbb{Q} \] and \[ \mathbb{Z} \times \mathbb{N^{\times}} \] \[ \blacksquare \]
