2.3. Quantifier and Property

2.3.1. Property

If \(E(x)\) is an expression which becomes a statement when x replace with a object or a specified class of objects then E is a property.

\(E(x)\) is true means “x has property E”

\(\{ x \in X; E(x) \}\) means every x in X has property E.

2.3.2. Quantifier

There has two quantifier:

  • \(\exists\): Exist at least one
  • \(\exists !\): Exactly one
  • \(\forall\): For all

and with property we write:

  • \(\forall x \in X: E(x)\)
  1. Order of Quantifier

    Order of quantifier is significant. For example, “\(\forall x \exists y: E(x,y)\)” and “\(\exists y \forall x: E(x, y)\)”.

    “\(\forall x \exists y: E(x,y)\)” means for all x there is some y such that E(x,y) is True.

    “\(\exists y \forall x: E(x, y)\)” means it suffices to find a fixed y such that E(x,y) is true for all x.

  2. Negation with Quantifier

    When apply negation to a statement which contain quantifier these quantifier should be interchanged between \(\forall\) and \(\exists\).

    For example:

    • \(\neg\neg A := \neg(\neg A) = A\)
    • \(\neg(A \land B) = (\neg A) \lor (\neg B)\)
    • \(\neg(A \lor B) = (\neg A) \land (\neg B)\)
    • \(\neg(\exists x \in X: E(x)) = (\forall x \in X: \neg E(x))\)

2.4. Implication

\((A \Rightarrow B) := (\neg A) \lor B\)

\(A \Rightarrow B\) is common to express as “To prove B is suffices to prove A” or “B is necessary for A to be true”, in other words, A is a suffice condition for B, and B is a necessary condition for A.

remark

  • a := b, means a is defined by b

2.5. Equivalence

\((A \Leftrightarrow B) := (A \Rightarrow B) \land (B \Rightarrow A)\)

So, \((A \Leftrightarrow B)\) is true when both \(A \Rightarrow B\) and it’s converse \(B \Rightarrow A\) is true, means A is suffice and necessary condition for B. Common way to expressing this is to say ’A is true if and only if B is true.

By observation:

  • \((A \Rightarrow B) \Leftrightarrow (\neg B \Rightarrow \neg A)\)

2.6. Others

\(A :\Leftrightarrow B\), say ’A is true, by definition, if B is true’
