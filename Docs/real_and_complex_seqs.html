  • \(\lim\) as a linear function
  • inner multiplication of convergent sequences
  • Convergent real sequences and the order structure of R. \(\lim x_n < \lim y_n\) .

2.1. Division on Term and it’s successor

Motivation: Build an inequality between \(x_{n+1}\) and \(x_{n}\), the inequality can used to reveal property of sequence, for example whether the sequence is monotone or whether is it less than an null sequence.

Try to see whether \(\frac{x_{n+1}}{x_n}\) is convergent or not. If it’s convergent then able to build an inequality:

\[ x_{n+1} < x_n\alpha \]

Or to prove that \(x_{n+1} \geq x_n\) (\(x_{n+1} \leq x_n\)).

Base on this inequality we then try to construct inequality that satisfy definition of convergence. An example is on Examples 4.2 (b).

2.2. Assumption on Index of Sequence

By make assumption on index of sequence the sequence may more easy to reveal that a sequence is convergent or not.

For example, \(\lim \frac{a^n}{n!} = 0\) where \(a \in \mathbb{C}\). By assumption on index \(n > N > |a|\) we are able to get the inequality, \(|\frac{a^n}{n!}| < |\frac{N^n}{N!}|\). More complete example in Examples 4.2 (c).
