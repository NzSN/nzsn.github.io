1. Order Completness

Every nonempty subset of R that is bounded has supremum and infinum.

2. Dedekind’s Construction of the Real Numbers

Function to embed \(\mathbf{Q}\) into \(\mathbf{R}\) is:

\[ \mathbf{Q} \rightarrow \mathbf{R}, r \mapsto \{ x \in \mathbf{Q} \mid x > r \} \]

Idea is to identifying the missing number by a pair of rational sets. The way to identifying the missing number is used a pair of rational set, for example, \(\sqrt{2} := \{x \in \mathbf{Q} \mid (x > 0) \land (x^2 > 2) \}\). Order completeness is a character expected as an extension field of \(\mathbf{Q}\).

Order on \(\mathbf{R}\):

\[ \mathbb{R_1} \leq \mathbb{R_2} := \mathbb{R_1} \supseteq \mathbb{R_2} \]

As we know, \(\subseteq\) on ses is Partial Order, but with the restrict of how those \(\mathbb{R}\) spawned we say this relation is totally ordered.

Order Completness is expected on \(\mathbf{R}\). Suppose that \(\mathbb{R_1} \subseteq \mathbf{R}\) is nonempty and bounded below, that is, there is some \(\mathbb{A} \in \mathbf{R}\) such that \(\forall r \in \mathbf{A}: r \subseteq \mathbf{A}\).

3. Square root

Prop. For all \(a \in \mathbb{R^+}\) and \(n \in \mathbb{N^{\times}}\), there is unique \(x \in \mathbb{R^+}\) such that xn = a.

Proof:

Uniqueness of solution can be derived from formula which already proved: \(a^n - b^n = (a-b)\displaystyle\sum_{k=0}^{n-1} a^{k}b^{n-k} > 0\)

Suppose \(x > a\) and have \(x^n > x > a\), assume that a > 1. Next to approximate the solution by declare a set that a as an upper bound: \(\mathbf{X} := \{ x \in \mathbb{R^+}; x^n < a \}\)

Supremum of \mathbf{X} should be solution of \(x^n = a\) I think, let’s to prove it. Let \(s_n := sup(X)\) and since \(\mathbb{R}\) is totally order so able to prove this by assume \(s^n > a\) or \(s^n = a\) or \(s^n < a\).

First to suppose that \(s^n < a\). Then we can find some \(\epsilon \in \mathbb{R}\) such that \(0 < \epsilon < (a - s^n)\), from this inequality have $sn + ε < a $. Intuitively, there should be real numbers between a and \(s^n\) but because s is sup(\mathbb{X}). Let to prove this is a contradiction. Let \(b := \displaystyle\sum_{k=0}^{n-1} \binom{n}{k} s^k\), and have \(0 < \epsilon < \frac{a - s^n}{b}\) able to assume \(\epsilon\) is small enought so that \(\epsilon < 1\). Then

\((s + \epsilon)^n = s^n - \displaystyle\sum_{k=0}^{n-1} \binom{n}{k} s^k \epsilon^{n-k} \leq s^n - (\displaystyle\sum_{k=0}^{n-1} \binom{n}{k} s^k)\epsilon < a\) which implies that \(s + \epsilon \in \mathbb{X}\) but which is impossible so \(s^n < a\) is not true.

Assume that \(s^n > a\). \(0 < \epsilon < s^n - a\). Similarly, to reveal contradiction that some real numbers between \(s^n\) and a so there are some upper bound least than \(s^n\). Let \(b := \displaystyle\sum^* \binom{n}{2j - 1} s^{2j - 1} > 0\) so \(0 < \epsilon < \frac{s^n - a}{b}\) with \(\epsilon \leq 1 \land s\). Thus we have

\begin{equation} \begin{align*} (s^n - \epsilon) & := \displaystyle\sum_{k=0}^{n} \binom{n}{k} s^n (-\epsilon)^{n-k} \\ & = s^n + \displaystyle\sum_{k=0}^{n-1} (-1)^{n-k}s^n \epsilon^{n-k} \\ & > s^n - \displaystyle\sum^* \binom{n}{2j - 1} s^{2j - 1} \epsilon^{n - 2j + 1} \\ & > s^n - (\displaystyle\sum^* \binom{n}{2j - 1} s^{2j - 1})\epsilon & > a \end{align*} \end{equation}

Now let \(x \in \mathbb{R^+}\) such that \(x > (s^n - \epsilon)\), and from above equation we have \(x^n > (s^n - \epsilon) > a\) which implies that \(s^n - \epsilon\) is an upper bound of \(\mathbb{X}\) which is impossible. Due to \(\mathbb{R^+}\) is totally order so \(s^n = a\).

Now to deal with the case the \(0 < a < 1\). Let \(b := \frac{1}{a}\), and there exists \(y \in \mathbb{R+}\) such that \(y^n = b\) then \(x := \frac{1}{y}\) is the solution of \(x^n = a\).

4. Supremum and Infimum

Prop.

  1. if \[ \mathbb{A} \subseteq \mathbb{R}\] and \[x \in \mathbb{R} \] then
    1. \[ x < sup(A) \iff \exists a \in A: a > x \]
    2. \[ x > inf(A) \iff \exists \in A: a < x \]
  2. Every subset \[ \bf{A} \] of \[ \mathbb{R} \] has a supremum and an infimum in \[ \mathbb{\bar{R}} \].

Proof. a.1. Assume that \[ x < sup(\mathbf(A)) \]. Suppose \[ \forall a \in \mathbb{A}: a < x \] then \[ x := sup(\mathbf{A}) \] which is contratry to assumption. Hence, \[ \exists a \in \mathbb{A}: a > x \]. a.2 Similar to a.1.

  1. If \[ \mathbf{A} \] is nonempty and bounded then claim is derived directly from order completeness. If \[ \mathbf{A} \] is nonempty and not bounded then \[ sup(\mathbf{A}) := \infty \] and \[ inf(\mathbf{A}) := -\infty \]. So the claim is true \[ \blacksquare \]

5. Archimedean Property

Prop. For \[ x \in \mathbb{R} \], there exists some \[ n \in \mathbb{N} \] such that \[ n > x \]. Proof. Let \[ x \in \mathbb{R} \] and \[ \mathbf{X} := \{ n \in \mathbb{N} \mid n < x \} \]. \[ s := sup(\mathbf{X}) \] then by (Prop Supremum and Infimum) there exists \[ a \in \mathbf{X} \] such that \[ s - 1/2 < a \] and let \[ n := a + 1\] so have \[ n > s \] so \[ n \notin \mathbf{X} \] Hence, \[ n > x \].

6. Density of Rational Numbers

Prop. For all \(a,b \in \mathbb{R}\) such that \(a < b\) there is some \(r \in \mathbb{Q}\) such that \(a < r < b\).

Proof. Because of \(a < b\) so \(b - a > 0\). By Archimedean property have \(n > \frac{1}{b-a}\), \(n \in \mathbb{N}\), so \(nb - na > 1\) and have \(nb > na + 1\). Archimedean property again, exists \(m_1 > na\) and \(m_2 > -na\) so \(-m_2 < na < m_1\). Consequence there exists \(m \in \mathbb{Z}\) such that \(m < na + 1 < m + 1\). Together with both of two inequality we have:

\[ na < m \leq na + 1 < nb \Rightarrow a < \frac{m}{n} \leq a + \frac{1}{n} < b \]

so exists some rational number \(\frac{m}{n}\) between arbiterary pair of rational numbers.

7. Density of Irrational Numbers

Prop. For any \(a,b \in \mathbb{R}\) such that \(a < b\), there is some \(\xi \in \mathbb{R} \backslash \mathbb{Q}\) such that \(a < \xi < b\). Idea is to construct an identity between a and b that can be proved is an irrational. As we known \(\sqrt{2}\) is an irrational. So can try to represent \(\xi\) by \(\sqrt{2}\).

Proof: A directly definition of \(\xi\) is \(a + \frac{b - a}{\sqrt{2}}\) but \(\sqrt{2} = \frac{b-a}{\xi-a}\) cannot gurantee that \(\xi\) is an irrational cause a and b may be irrational too so \(\xi\) can be rational in this equation.

Fortunately, density of rational numbers can help. By density of rational numbers we have \(a < r_1 < b\) and \(a < r_2 < b\) such that \(r_1 < r_2\). Hence, \(\xi := r_1 + \frac{(r_2 - r_1)}{\sqrt{2}}\). Clearly, \(a < r_1 < \xi < r_2 < b\) and \(\xi\) is irrational due to \[ \sqrt{2} = \frac{r_2 - r_1}{\xi - r_1} \]
