1. \(x_nx_{n+1}\) about mean Transformation

A way to transform \(x_nx_{n+1}\) from \(x_n + x_{n+1}\).

1.1. Scenario

\(\alpha := \frac{x_1 + x_2 + ... + x_n}{n}, n\alpha = x_1 + x_2 + ... + x_n\)

This transformation require that \(\alpha\) is an element of an order field.

When some order relation is exists between \(n\alpha\) and \(x_1x_2...x_n\). If you want assert that those relations is true for \((n+1)\alpha\) and \(x_1x_2...x_{n+1}\) then you can use this transformation.

1.2. How

\(\alpha^n \leq x_1x_2...x_n\) \[(x_n - \alpha)(\alpha - x_{n+1}) = (x_n + x_{n+1} - \alpha)\alpha - x_nx_{n+1}\] \((n+1)\alpha = x_1 + x_2 + ... + x_{n+1}\) \[n\alpha = x_1 + x_2 + ... + x_n + x_{n+1} - \alpha\]

By these equations we have

\[\alpha^{n+1} = (\alpha^{n} \alpha) \leq (x_1x_2x_3...x_{n-1}(x_n + x_{n+1} - \alpha)\alpha)\]

2. Application of Archimedean Property

With Archimedean Property we are able to assume two Naturals that enclose a specific Real Number.

2.1. Scenario && How

Let \[ x \in \mathbb{R} \] then with this property we have \[ x < n_1 \] and \[ -n_2 < x \] so \[ -n_2 < x < n_1 \]. Consequence have exists \[ n \in \mathbb{N} \] such that \[ n - 1 < x < n \].
