2. What are operators ?

Operators are functions. There are two kinds of operators:

Operators are functions that basis is Observable.

2.1. Pipeable Operators

A Pipeable Operator is a function that takes an Observable as its input and returns another Observable. It is a pure operation: the previous Observable stays unmodified.

Work like a trsformers.

2.2. Creation Operators

Creation Operators are the other kind of operator, which can be called as standalone functions to create a new Observable.

3. Piping

A way to convolve Pipeable Operators.

Example: 

obs.pipe(
  op1(),
  op2(),
  op3(),
  op3(),
)

4. Higher-order Observables

Observables of Observables. 

const fileObservable = urlObservable.pipe(
   map(url => http.get(url)),
);
