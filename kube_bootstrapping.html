2.1. Initializing your control-plane node

2.2. How recover after node reboot ?

2.2.1. Master node recover after down

Just run: 

systemctl start kubelet

you can set kubelet auto start by 

systemctl enable kubelet

2.2.2. Windows worker node recover after down

Run: 

docker network create -d nat host

2.3. Join Nodes

kubeadm join --token <token> <control-plane-host>:<control-plane-port> --discovery-token-ca-cert-hash sha256:<hash>

token is able to get by 

kubeadm token list

ca-cert-hash is able to get by 

openssl x509 -pubkey -in /etc/kubernetes/pki/ca.crt | openssl rsa -pubin -outform der 2>/dev/null | \
   openssl dgst -sha256 -hex | sed 's/^.* //'
