2.1. How to build up a set ?

2.1.1. Inductive definitions

Inductive definition is consist of a series of inference rules.

There is a concrete syntax:

Constants n \(\in\) \(\mathbb N\) Variables x \(\in\) \(\mathbf Strings\) Expressions e ::= n | x | e + e | e \(\times\) e

This BNF grammar implicitly giving an inductive definition.

A more general notation for inductive definitions provides a series of inference rules that define a set:

\(\frac{n \in \mathbb{N}}{n \in \mathbf{Exp}}\) \(\frac{x \in \mathbf{Strings}}{x \in \mathbf{Exp}}\) \(\frac{e_1 \in \mathbf{Exp}, e_2 \in \mathbf{Exp}}{e_1 + e_2 \in \mathbf{Exp}}\) \(\frac{e_1 \in \mathbf{Exp}, e_2 \in \mathbf{Exp}}{e_1 \times e_2 \in \mathbf{Exp}}\)

These four rules that together are equivalent to the BNF grammar above.

2.1.2. Structural Induction Principles

Structural Induction is similar to Mathmatical Induction, cause able to viewing a natural numbers as one simple inductive definition set.

  1. How to get Induction Principle from Induction Definition ?

    When we define a set S inductively, we gain an induction principle for proving that some predicate P holds for all elements of S.

    We are able to derive an induction principle via:

    • For each \(E \in S\) on conclusion, replace with \(P(E)\).
    • For each \(E \in S\) on premise, add a companion premise \(P(E)\). That is, the obligation allows assuming that P holds of certain terms. Each such assumption is called an inductive hypothesis (IH).

2.1.3. Decidable Theories

A theory is a set of formulas, often assumed to be closed under logical consequence. Decidability for a theory concerns whether there is an effective procedure that decides whether the formula is a member of the theory or not.

In logic, a true/false decision problem is decidable if there exists an effective method for deriving the correct answer.

Undecidable, that is, it has been proven that no effective method for determining membership (returning a correct answer after finite, though possibly very long, time in all cases) can exist for them.

Decidability of theories is handy because, whenever our goal belongs to the F set of a decidable theory, we can discharge the goal automatically by running the deciding program that must exist.
